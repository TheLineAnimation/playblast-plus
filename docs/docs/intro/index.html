So what is it?

Built Withâ€‹

Hostsâ€‹

Playblast Plus is a python library that is designed to facilitate animation workflows with consistent, yet customisable content previews in a variety of DDC Hosts.

Let's face it, Maya's playblast isn't the best. (And that's saying something as a long-time 3dsmax user) This script standardizes the playblast output to allow animators an easy way to create a rolling, local playblast to review their work.

tip

You can also click the header in the UI to launch this help inside your DCC.

As an animator, you don't always want to submit a playblast for project review.

There might be a number of iterations you need to perform before you are ready to let those keyframes fly, like little tweened starlings flocking in an animated sky.

But Why?

Both Maya and 3dsMax have pretty awful ways to create animation previews.
They aren't particularly sharable, and animation sequences have very sketchy codec support.

What it doesâ€‹

Playblast Plus streamlines the animation preview process by:

  • Leveraging the host application's native preview tools to generate frame sequences, then automatically converting them to H264 MP4 files using FFmpeg. This enables animators to quickly review their work without lengthy rendering times.
  • Applying customizable view templates to ensure all previews maintain a consistent look, so reviews from different animators share identical visual settings.
  • Allowing flexible overrides for parameters such as image planes (backgrounds), wireframe rendering, and isolating key components, giving artists control over their previews.
  • Supporting both full animation previews and single viewport images for versatile review options.

Featuresâ€‹

  • Transcodes the Maya playblast into MP4 (Uses FFMpeg)
  • Template overrides for common requirements, like show wireframe and image planes
  • Tokenized path output to control filenames and paths.

Installationâ€‹

To install, download the latest release and place the playblast_plus folder in a directory on your PYTHONPATH

Usageâ€‹

To show the interface in any supported host, run:

from playblast_plus import launch as pbp
pbp.run()
